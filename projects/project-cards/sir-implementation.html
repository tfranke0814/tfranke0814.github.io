SIR Model Implementation

This was my final project for Math/CS 514: Numerical Analysis at UW-Madison during the Fall 2025 semester. The SIR model is a mathematical model of epidemic outbreaks. It has been the cornerstone of mathematical epidemiology.

I wrote MATLAB code to implement the system of ODEs described by the model using the fourth-order Runge-Kutta method. I followed with a quick writeup about the SIR model, discussing the implementation and the results. After settling on the method, I played around with the parameters to simulate different scenarios. Included are some graphs and their interpretations.
